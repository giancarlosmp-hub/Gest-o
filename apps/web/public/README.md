# Public assets required for branding

Add this file for the favicon used by `index.html` and `manifest.webmanifest`:

- `apps/web/public/favicon.png`

Build output expectations:

- `apps/web/dist/favicon.png`
- `apps/web/dist/manifest.webmanifest`
- `apps/web/dist/brand/demetra-logo-dark.png`
- `apps/web/dist/brand/demetra-logo-light.png`
